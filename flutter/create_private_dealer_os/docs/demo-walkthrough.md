## Walkthrough script (8-10 min)

1. Login
   - Sign in and confirm role-based tabs.
2. Dashboard
   - Highlight New Leads, Avg Response, Bookings, Missed Leads.
3. Conversations
   - Open a thread, update lead status, assign lead, create booking, send message.
4. Leads
   - Apply filters by status/source/assigned.
   - Open lead detail and show followups/recent messages.
5. Bookings
   - Create booking.
   - Change booking status (booked/completed/no_show/cancelled).
6. Settings (admin)
   - Edit business hours and FAQs.
   - Toggle marketing message visibility.
   - Invite and disable a user.
7. Analytics
   - Switch 7/30 day response trend.
   - Show leads-by-source bars and summary tiles.
8. Privacy and security checks
   - Show phone masking.
   - Confirm message sending is backend API mediated.
